# Chat Widget Iframe App

React-based chat UI for the iframe component.

## Features
- Message rendering (text, images, buttons, forms)
- Streaming responses
- Theme support via CSS variables
- PostMessage communication with loader

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

Outputs to `dist/` for deployment.
